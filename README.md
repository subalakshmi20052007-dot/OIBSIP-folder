# Brew Haven Coffee Website ☕

## Project Overview
Brew Haven Coffee is a responsive coffee shop website designed to provide an attractive online presence for a coffee brand. The website showcases coffee products, menu items, gallery images, customer reviews, and contact information.

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Features
- Attractive home page with hero section
- Coffee menu section
- Image gallery
- Customer review section
- Contact information
- Responsive design for different screen sizes

## Purpose
The main purpose of this project is to create a user-friendly and visually appealing coffee shop website using front-end web technologies.
